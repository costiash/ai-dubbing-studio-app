## User Journey Overview

### Primary User Flow

```
Upload Audio â†’ Transcribe â†’ Review/Edit â†’ Translate â†’ Generate TTS â†’ Download
    |            |            |            |              |            |
  (Step 1)    (Step 2)     (Step 3)     (Step 4)      (Step 5)    (Step 6)
```

### Time Estimates

| Step | Action | Estimated Time |
|------|--------|----------------|
| 1 | Upload audio file | 2-10 seconds (depending on file size) |
| 2 | Transcribe audio | 30-60 seconds (API processing) |
| 3 | Review/edit transcript | 2-5 minutes (user editing) |
| 4 | Translate text | 10-20 seconds (API processing) |
| 5 | Generate TTS audio | 15-30 seconds (API processing) |
| 6 | Download result | Instant |

**Total:** ~5-10 minutes for complete workflow
